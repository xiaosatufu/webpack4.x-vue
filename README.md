### 安装webpack webpack-cli
cnpm install webpack webpack-cli -g
查看webpack webpack-cli版本
webpack -v
webpack-cli -v

### 新建package.json

```
npm init -y
```
### 安装vue webpack webpack-dev-server
```
npm install vue --save-dev
npm install webpack webpack-dev-server
```
